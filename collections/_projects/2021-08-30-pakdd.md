**Full-Stack · UI/UX · Static Architecture**

Designed and developed a conference website from scratch, creating both the visual identity and a maintainable content platform for future organisers.

- Defined a new **visual identity and design system**, translating it into responsive UI through Figma and front-end implementation
- Built the site using **Jekyll and Bootstrap**, structuring content to be easily managed via Markdown without developer involvement
- Added **interactive elements** using p5.js to enhance engagement beyond static content
- Implemented a **custom authentication system** using Firebase, enabling secure access to internal documents and resources for committee members

- Designed the system to be **maintainable by non-technical stakeholders**, reducing long-term overhead
- Balanced **custom functionality (auth, interactivity)** with a lightweight and scalable static architecture

_Focused on building a complete, maintainable web platform from design through to deployment._
