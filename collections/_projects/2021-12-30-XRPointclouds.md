Developed an XR system for visualising and interacting with large-scale point cloud data on HoloLens 2.

- Built support for **.ply point cloud files** in Unity, enabling accurate rendering of complex 3D datasets in XR
- Implemented **interaction techniques** (grabbing, scaling, rotating) for intuitive exploration of spatial data
- Designed a **cloud-based pipeline** to stream point cloud data from desktop systems to the HoloLens
- Enabled users to **browse, fetch, and visualise datasets** dynamically via a Firebase-backed API

- Addressed challenges around **large file handling and performance** in resource-constrained XR hardware
- Decoupled data storage and visualisation, supporting more flexible and scalable workflows

_Focused on making complex 3D data accessible and explorable within constrained XR environments._
