Investigated how subtle manipulation of rotational feedback in VR affects users’ sense of presence, spatial awareness, and proprioception.

- Developed a Unity-based VR experiment where users performed **object interaction and memory tasks** within a controlled virtual environment
- Introduced **imperceptible rotational offsets** to the virtual scene, creating a mismatch between user expectation and system response
- Designed the system to evaluate how these manipulations impact **spatial understanding and task performance**

- Explored the relationship between **perceived motion and physical orientation**, and how discrepancies influence immersion
- Investigated how controlled sensory mismatch can be used to **alter user perception without explicit awareness**
- Observed changes in user spatial perception and task performance under manipulated rotational feedback

_Focused on understanding how subtle sensory manipulation can influence perception and presence in immersive environments._
