**XR Systems · Data Pipelines · Cloud Integration**

Built a system for **live transfer of graph data** from desktop analytics environments (Python/Mathematica) into immersive XR applications.

- Developed a **cloud-based pipeline** enabling real-time synchronisation of 3D graph data between desktop notebooks and a Unity XR client
- Built an XR application that **consumes and visualises live data** via a Firebase-backed API
- Created custom **Python/Mathematica tooling** to generate and stream graph data directly from existing analysis workflows
- Designed the system to leverage **mature desktop analytics tools** while enabling immersive exploration in parallel

- Decoupled **heavy computation (desktop)** from **real-time visualisation (XR)** through a modular architecture
- Enabled **distributed and remote usage**, supporting expert evaluation in fully virtual settings

_Focused on bridging traditional data science workflows with immersive, real-time visualisation systems._

Published as a poster at _IEEE VR 2022_.
