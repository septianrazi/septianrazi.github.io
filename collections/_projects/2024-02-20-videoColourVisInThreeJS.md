Developed an interactive system for visualising video colour distributions across multiple colour spaces, translating video pixel data into spatial 3D representations.

- Processed video input to extract **per-frame colour data**, aggregating it into density-based visualisations
- Implemented **WebGL shaders** to dynamically generate and manipulate geometry for rendering colour distributions
- Visualised colour in **RGB, CIExyY, and CIELab spaces**, enabling comparison between different perceptual models
- Built interactive 3D graphs in Three.js, with **WebXR support** for immersive exploration of colour data

- Explored how different colour spaces impact **perception and interpretation of visual data**
- Leveraged GPU-based rendering to handle **large-scale point/geometry transformations in real time**

_Focused on making abstract colour data more interpretable through spatial and interactive visualisation._
